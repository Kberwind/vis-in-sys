# visual-interactivesystems
Data analysis and reporting for Seminar on Visual and Interactive Systems and TU Graz.

This repository contains

A locally hosted html based website containing the data analysis for this seminar project. Access it from _site and opening the index.html file
The excel sheet used for content analysis of open-ended questions
The final report of findings and design considerations
The presentation slides

This repository does not include the dataset, as the whole dataset is not publicly availible.
